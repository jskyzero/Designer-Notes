這本書就相對比較嚴謹的從一門專業課程的角度去敘述了遊戲的來龍去脈，這裡也打算相對嚴謹一點的複述其中較為重要的部分。

<!--more-->

## 前言

「遊戲」是什麼？你或許看到次時代主機上最新大作逼真美麗的畫面，或許在手機程序上或大或小的世界中打發過時間，或許和熟人連麥一起去頂尖的副本冒險，遊戲的過去是什麽樣子，現在是什麼情況，未來又可能向著什麼方向去發展呢？

接下來本文將按照《遊戲專業概論》的框架，試著去梳理一些較為基礎同時較為重要的部分。

## 正文

### 遊戲起源

1. 遊戲的概念
  + 遊戲的定義：思想家，柏拉圖，亞里士多德等都曾給出自己的定義，辭海中給出的定義指出了兩個基本特性——獲得快感和主體參與，綜合以上觀點可以得到：1.遊戲是自願的活動。2.遊戲包含著豐富的快樂體驗。3.遊戲是有規則的活動。莫利斯在《裸狼》中曾經將人類活動去除次要目的并加以限制理解成遊戲，他的觀點可以綜合成：遊戲是以重複和變化的手段進行探索的活動。可以說，探索是生命的本性，遊戲也是生命的本性。
  ![]({{site.img_url}}/books/introduction_to_game/introduction_to_game (1).jpg)
  + 遊戲相關的理論：1.康德的自由論，強調與被迫勞動相對立的自由。2.席勒的和諧論，感性和理性的和諧統一。3.弗洛伊德的虛擬論，“遊戲是人藉助想象來滿足自身願望的虛擬活動”。4.胡伊青加的內在目的論，主張用特征描述的方式來把握遊戲定義。5.伊達默爾的融合論，說遊戲是以自我表現為內在目的的自為活動。
  + 遊戲的本質：遊戲的特性有，1.體驗性。2.規則性。3.自主性。4.虛幻性。
  ![]({{site.img_url}}/books/introduction_to_game/introduction_to_game (2).jpg)
2. 遊戲的發展與演變
  + 最初的“玩”是無意識、無規則的本能表現，訓練自己的生存本能，掌握了生產勞動技能且有剩餘勞動力以後，生產便向娛樂活動轉變，雖然受到歷史與文化的差異，但是遊戲是人類文化發展的必然產物。
  + 舉例來說，1.1976年，許家窯村發掘出的石球從武器變為遊戲工具。2.3000多年前呂底亞人用擲骰子熬過漫長的18年饑荒。3.古希臘、古羅馬的遊戲起源於宗教祭祀活動。傳統遊戲可以分為競技、思維斗智、猜謎、賭博等。
  + 工業革命使得規則意識和商業化改變了社會，而當今社會的信息時代，網絡遊戲的互動、虛擬等特征也符合了新時代的發展要求。每個時代遊戲的精神內核在不斷變化，但基礎都是需要符合社會文化需求和發展的需要。
3. 新時代遊戲的定義
  + 電子計算機的出現標誌著人類社會正式步入計算機時代，人類的遊戲行為也從傳統遊戲發展過渡為電子遊戲，當前時代的遊戲通常指在各種電子設備平台下運行的遊戲應用程序，遊戲的類型也有所不同。究其本質而言，遊戲的核心意義仍然是在工作、勞動之餘尋求快樂的一種主動行為活動。
  + 遊戲中常見的樂趣元素：1.角色扮演。2.對操作感的需求。3.社交層面。4.挑戰和成就意識。5.收集嗜好。6.探索屬性。7.個性化自定義。8.故事沉浸。
  ![]({{site.img_url}}/books/introduction_to_game/introduction_to_game (3).jpg)
4. 遊戲心理學
  + 心理學是一門研究人類及動物的心理現象、精神功能和行為的科學，遊戲心理學就是利用心理學原理來研究和解釋人類的遊戲行為。屬於應用心理學的範疇。
  + 從經典的俄羅斯方塊的選擇分析可以得到經典的馬西米尼的心流體驗模型，高挑戰與高技能水平會產生心流，此時人的身心處於最積極、意識處於最享受的狀態。
  + 如何製造持續的心流體驗？芬蘭學者凱利曾提出一個體驗式的遊戲學習模型，由構思迴路、體驗迴路和挑戰池組成。告訴了我們關鍵在於有一個完善的循環學習架構。
  + 现实遊戲，PVP需要有一個合適的水平評估算法和對手匹配機制，PVE類型的遊戲則需要嚴格按照心流循環架構來構建佈置關卡。
  + 在遊戲的美術設計上，美術原色的形狀、體積、尺寸都會對玩家的心理產生不同的影響。有很多具體且現實的例子，合理運用遊戲心理學的理論，不僅會增加遊戲的樂趣和可玩性，也會從根本上抓住玩家的心理，讓遊戲產品的壽命更加長久。


### 電腦遊戲發展史

1. 電腦遊戲的誕生
  + 1942年2月，出於對彈道研究的需要，世界第一台計算機在美國賓夕法尼亞大學誕生，簡稱ENIAC，在其基礎上誕生了第一台民用計算機UNIVAC，標誌著計算機從實驗室走向了大眾。
  ![]({{site.img_url}}/books/introduction_to_game/introduction_to_game (4).jpg)
  + 1958年，計算機仍然是高級計算工具，為了讓參觀的人能多參觀一會兒，Tennis for Two誕生了。
  + 1962年，在DEC公司PDP-1小型機上，誕生了第一個真正意義上的遊戲程序——《Space War》，標誌著數字化遊戲形式的正式誕生。
2. 2D游戲時代
  + 受到早期計算機硬件的限制，早期的游戲只能用像素顯示圖形，1976年，Apple Ⅰ 誕生，告訴了世界什麽是個人電腦，次年 Apple Ⅱ 開始正式商業化銷售。
  + 第一款PC游戲出現在1978年，是一款純文字冒險游戲《冒險島》，1979年，第一款RPG游戲《巫術》（wizardy）誕生，1980年，運行在Apple Ⅱ平臺的《創世紀》（Ultima）發售，在當時游戲玩家眼中被視爲游戲史上的“聖經”。1986年，已經具備後期系列作元素的《魔法門》一代誕生。這三個游戲一開始就決定了歐美RPG的超高自由度。
  + 1981年，IBM推出搭載微軟DOS1.0的開放性架構的IBM 5150 PC，隨著硬件水平的提高，更多類型的游戲出現。1884年，PC第一款圖像冒險類游戲《王國密使》（King's Quest）推出，1989年，PC上第一款原生ACT游戲《波斯王子》誕生。同樣是1989年，模擬類游戲《模擬城市》推出。
  + 這一時期還有很多很多其他經典的游戲系列推出，比如AVG的典型代表《猴島小英雄》、《鬼屋魔影》、《神秘島》；ACT的經典作品《決戰富士山》、《雷曼》；SLG的著名游戲《三國志》、《文明》；RTS的開山之作《魔獸爭霸》。
  + 隨著硬件的進一步升級，里程碑式的Win95操作系統，Wintel聯盟，DirectX API的推出給了《暗黑破壞神》絕佳的畫面，1991年，文明推出，1992年，id software推出歷史上的第一部FPS游戲《德軍總部3D》。1993年加入了真實時間的《Dune Ⅱ》標志著RTS游戲的誕生，時間因素使得其和SLG游戲劃開了界限，1995年，擁有聯網功能的《命令與征服》誕生，還有了名爲《紅色警戒》的資料篇。
  ![]({{site.img_url}}/books/introduction_to_game/introduction_to_game (7).jpg)

3. 3D游戲時代
  + 1996年6月，id software的QUAKE是PC進入3D時代的標志。依靠3dfx創造的Voodoo——PC歷史上的最經典的一款3D加速顯卡，把PC拉入了瘋狂的3D時代，隨機第一款正式支持它的游戲就是《古墓麗影》，微軟最終決定統一各種圖像芯片的接口推出DirectX ，NVIDIA全力支持DirectX和OpenGL並最終成爲當時顯卡市場的唯一霸主。
  + 1996年到2000年，Intel的Pentium（奔騰）處理器的性能也在不斷提升，1997年的《暗黑破壞神》確立了一個全新的游戲類型ARPG，同時引入了2.5D的概念，同年，超高自由度的輻射誕生。對後世同樣產生深遠的影響。1998年的《星際爭霸》代表了RTS對戰的興起，從DOOM開始的FPS的對戰游戲也發生了變化，產生了强調動作、劇情、設計的第三個分支。1999年反恐精英發佈，在全世界取得了極大的歡迎。
4. 網絡游戲時代
  + 聯機對戰游戲打破了固定程序模式的束縛，早期的網絡游戲主要是RTS和FPS，RTS比如《帝國時代Ⅱ》、《家園》、《魔獸爭霸Ⅲ》，FPS比如《Quake Ⅲ》、《反恐精英》等等，這些游戲普遍强調操作的熟練度與快速反應度。
  + 隨後出現了强調參與感、體驗感、社交性的大型多人在綫角色扮演游戲（MMORPG），早期就曾經出現過包含這些特性的純文字游戲，最早的圖形化界面MMORPG是1997年的《創世紀網絡版》（Ultimate OL），它也是第一個全球化的游戲。
  + 經過漫長的發展，現在歐美、日韓、中國各自有各自的特點和現狀。

### 遊戲機發展史

1. 游戲機的誕生
  + 目前所說的游戲機主要是指專門運行虛擬游戲程序的硬件主機設備，1971年，世界上第一台游戲機Computer Space誕生，它比較類似現在的街機游戲機。雖然最後失敗，但是它的某位開發者參與成立了雅達利公司。
  ![]({{site.img_url}}/books/introduction_to_game/introduction_to_game (5).jpg)
  + 1967年，“電視游戲”改革計劃的提出，1972年，名爲“Odyssey”（奧德賽）的家庭游戲主機正式上市，當時為其製作游戲的就是雅達利公司，一款名爲Pong的乒乓游戲就是世界上第一款被公衆接受的街機游戲。
  ![]({{site.img_url}}/books/introduction_to_game/introduction_to_game (8).jpg)

2. 雅達利的興衰
  ![]({{site.img_url}}/books/introduction_to_game/introduction_to_game (6).jpg)
  + Pong的成功使得雅達利占據著當時電子游戲市場的絕對主導地位。1974年，雅達利推出了自主研發的Pong家庭版游戲主機，1975年，已經售出了15萬套，1976年，雅達利發行過一個名爲《夜晚駕駛者》的3D主視角的街機游戲，1977年，雅達利自主研發的類似家用電腦的2600型家用游戲機上市，可以運行上百款游戲，取得前所未有的成功。
  ![]({{site.img_url}}/books/introduction_to_game/introduction_to_game (9).jpg)
  + 1982年末，人們最初對電子游戲的熱情開始減弱，有評論人士認爲是市面上大量的劣質游戲造成的，終於在聖誕節出現大崩潰，於此同時，雅達利公司巨資購買E.T.的版權沒能成功，推出的不兼容上代產品的5200型也沒能扳回局面，1983年，雅達利公司停止了股票交易，從此一蹶不振，最終在2013年申請破產。

3. 任天堂的興起
  + 任天堂的前身成立於1889年，最初是製作日本手製紙牌的小作坊，1965年，開始進軍電子產業，最初有一些玩具上的嘗試，1975年看到雅達利的設備后，與其他公司合作在1977年推出了自己的電視游戲平臺ColorTVGame6，在當時賣出數百萬台。
  ![]({{site.img_url}}/books/introduction_to_game/introduction_to_game (10).jpg)
  + 1978年，任天堂將目光投向了街機市場，製作了數款街機游戲，1979年，任天堂開始設計便携式電子游戲平臺Game & Watch，1980年正式上市，1981年的大金剛大賣，并讓任天堂看到了虛擬角色的價值。
  + 1983年，任天堂發佈了歷史上最經典的一款游戲主機 Family Computer（簡稱FC），由於主色是紅色和白色，也被稱爲紅白機，上市後取得了巨大的成功。
  + 之後SFC、N64、GameCube、Wii、NSW等家用主機問世，任天堂也締造了《薩爾達傳説》、《超級馬里奧兄弟》、《口袋妖怪》等經典電子游戲系列。
  ![]({{site.img_url}}/books/introduction_to_game/introduction_to_game (11).jpg)


4. 家用游戲機市場的群雄割據
  + 自FC發售以來，一切其他的公司也加入到了競爭中來，比如世嘉、NEC等。1986年世嘉公司推出了Sega Master System游戲機，1987年，NEC推出了最强的八位機PC-ENGINE，1988年為其推出了CD-ROM系統，大肆吞食任天堂的市場份額，1988年世嘉推出了世界第一臺16位游戲機Mega Drive，在美國市場獲得認可，最終銷售出3500多萬台。
  + 1990年，任天堂的16位家用游戲機SFC發售，在日本繼承FC的市場，但美國推出被世嘉擊敗，同年，日本SNK推出了Neo-Geo的家用游戲主機，定位是家用版街機，售價昂貴。
  + 1986年以後的這段競爭時期，誕生了非常多著名的作品，比如：1. 1990年SFC《超級馬里奧世界》。2. 1991年SFC《最終幻想Ⅳ》。3. 1992年《超級馬里奧賽車》。4. 1994年SNK的《拳皇》。5. 1996年Neo-Geo的《合金彈頭》。
  + 1992年，由EA牽頭，行業巨頭共同成立3DO公司，負責内容製作和銷售的工作被交給了松下公司，1993年推出了32位游戲家用機REAL 3DO INTERACTIVE MULTIPLAYER，雖然機能突出，但是由於市場運作和傳統游戲機不同等因素最終3DO解體，1994年越來越多的游戲開始嘗試3D，1994年世嘉發佈Sega Saturn，11天后，索尼發佈PlayStation，因爲對第三方廠商優惠的政策，SQUARE的《最終幻想Ⅶ》、《最終幻想Ⅷ》為其贏得了日本和世界的市場。
  ![]({{site.img_url}}/books/introduction_to_game/introduction_to_game (12).jpg)
  + 1995年，任天堂的32位主機Virtual Boy投放市場，有著頭戴式液晶眼鏡用作顯示器，買點也在立體游戲上，但當時畫面顯示素質確實有限，最終成爲了任天堂壽命最短的一款產品，1996年，任天堂的64位家用游戲機Nintendo 64上市，但是由於卡帶的成本缺乏游戲軟件的支持，由於任天堂公司種種錯誤的決策，自身一直在走下坡路。
  + 1998年，世嘉Dreamcast發佈，但是游戲軟件缺乏最終導致世嘉退出家用游戲主機領域，這一時期世嘉的游戲作品有比如《莎木》、《VR戰士》等等。

5. 電子游戲領域的“三國時代”
  + 2000年，索尼發佈PS2，2001年，任天堂發售Nintendo GameCube，首發游戲陣容過於單薄，任天堂的游戲製作品質也出現大幅下滑。2001年，微軟XBOX發售，强悍無比的硬件組合和低廉的售價，以及優秀的網絡游戲支持為XBOX贏得了市場，這段時期，2001年《最終幻想Ⅹ》，世嘉的《夢幻之星Online》，2003年《HALO》。
  ![]({{site.img_url}}/books/introduction_to_game/introduction_to_game (14).jpg)
  + 新一代主機爭奪戰由XBOX360率先拉開，索尼匆匆推出的PS3一直處於弱勢，任天堂又發佈Wii全新體感游戲機。
  ![]({{site.img_url}}/books/introduction_to_game/introduction_to_game (13).jpg)

6. 便携式掌機的發展
  + 1976年，世界上第一台帶有電子屏幕的掌機由美國Mattel推出，1979年，日本BAIDAI推出了太空侵略者LED屏幕掌機，1980年任天堂發佈Game & Watch掌機。1982年和1983年，任天堂還發售了Green House和Donkey Kong2的特殊掌機，是現在DS系列掌機的雛形概念。
  + 1989年任天堂推出便携游戲機GameBoy，點陣式液晶屏和掌機卡帶都是GB帶給世界掌機的巨大革命，往後雅達利、世嘉、NEC、日本SNK、BAIDAI都有推出過掌機。任天堂也與2001年發佈GBA掌機。
  + 2004年索尼發佈PSP，任天堂也發佈NDS，雖然初期效率被PSP全面壓過，但最終靠著雙屏幕的創新理念和衆多新穎的游戲軟件在之後的競爭中銷量逐步上升。
  + 2011年任天堂推出3DS，具有全新的裸眼觀察立體畫面的功能，索尼也與同年發佈PSV，引入應用商店等概念，但在游戲軟件的競爭中，索尼這次輸給了任天堂。
  ![]({{site.img_url}}/books/introduction_to_game/introduction_to_game (15).jpg)

7. 手機游戲的發展
  + 主要可以分爲Java平臺時代和智能手機時代。

### 遊戲分類
1. 平臺：電腦、街機、家用機、掌機、手機等
2. 載體：軟盤、卡帶、光盤、數字等
3. 内容：類型環概念，雛形，發展，確立，融合。可以具體劃分到：RPG、ACT、AVG、SLG(simulation)、SIM(simulation)、STG(shooting)、SPG(Sports)、RCG(Racing)、PUG(Puzzle)、MUG(Music)、Card Game、Online Game。


### 遊戲的研發以及市場運作

可以將游戲公司劃分爲：市場部、研發部、管理部等。

研發部又分爲測試與製作，製作包括策劃、程序、美術。

### 遊戲策劃

1. 需要相當高素養，包括：對市場敏銳的洞察力，一定的商業營銷頭腦，善於把握玩家心理、對游戲其他部門的工作有所瞭解、具備良好的藝術修養、具備一定的想象力和數理能力、具備良好的語言文字表達能力、善於采納別人的合理化建議、對游戲的熱愛。
2. 職能與作用
  + 主策劃：給下屬執行策劃分配任務，負責與程序、美術部分溝通，負責策劃設計工作的質量監督，負責項目進度安排、游戲整體設計架構等方面的工作。
  + 關卡策劃：掌握專業游戲引擎來設計關卡，同時需要配合數值策劃設定數值，配合劇情策劃進行設計，甚至系統設計等方面，需要跟程序部門協調配合，進行任務系統方面的實現，同時提出任務、場景編輯器等方面的需求，提交美術資源的需求並跟進。需要有一定美術基礎，對歷史、地理、建築等方面有一定瞭解。
  + 數值策劃：負責游戲中各種數據和數值的平衡設定，又稱爲游戲平衡性設計師，一般主要負責游戲平衡性方面的規則和系統的設計，負責游戲中各種公式運算的設計，整個經濟系統的搭建以及戰鬥系統的設計等等。需要嚴密的邏輯思維和數理能力。
  + 系統策劃：負責一些游戲系統規則的編寫，和程序設計者的工作比較緊密，對邏輯思維能力要求很高。如果是程序出生轉的話，轉系統策劃可能會比較容易。
  + 劇情策劃：一般負責游戲中的文字内容的設計，包括但不限於世界觀架構、主綫、支綫任務設計、職業物品説明、局部文字潤色等等，需要一定會的文學功底。
  + 其他執行策劃：例如UI策劃，脚本策劃，創意策劃，任務策劃，音樂策劃等等。
3. 一些其他有意思的東西
  + 好的關卡：運行流暢、難度合理、趣味性、耐玩性、視覺效果、新手指、創意。
  + 游戲策劃文檔：市場情況分析、游戲的大體介紹、盈利模式、細分模塊。

### 遊戲美術

1. 游戲美術的發展
  + 從程序繪圖時代到需要專門的人員，藉助專門的二維圖像繪製軟件，利用藝術修養和美術功底來完成游戲圖像的繪製工作。
  + 1996年，隨著3D顯卡的出現，游戲美術需要更多的人員和部門通力協作。
2. 游戲美術的風格和分類
  + 題材，幻想風格、寫實風格、Q版風格；
  + 畫面類型，像素，2D，2.5D，3D。
3. 職能劃分
  + 游戲美術原畫、二維美術設計師、三位美術設計師、特效美術師、地圖編輯美化師。

### 遊戲引擎
1. 定義
  + 起控制作用的代碼。
  + 需要包含：光影效果、動畫、物理系統、碰撞檢測、渲染、處理輸入。
2. 發展歷史
  + 誕生：1991-1993，《德軍司令總部》約翰卡馬克，《毀滅戰士》，《雷神之錘》，只有把自己的引擎交給給多人去使用才能使游戲引擎不斷成熟和發展起來。
  + 發展：1994-1997，Build引擎，Quake引擎，Unreal引擎。
  + 革命：1998-2000，半條命對引擎的影響，脚本序列技術，對AI人工智能的改進。一些歷史的上的引擎。
3. 主流引擎介紹
  + Unreal、CryEngine、Frostbite、GameBryo、BigWorld、id Tech、Soure、Unity3D。
